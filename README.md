# Python-Personal-Assistant
Skillcrush final 101 challenge for Python

This personal assistant app can help you keep track of all your to-dos, birthdays, and contacts.

How can I help you?

    **** To-dos *****
    1: Add a to-do
    2: Remove a to-do
    3: Get to-do list
    **** Birthdays ***** 
    4: Get Birthday
    5: Add Birthday
    6: Remove Birthday
    **** Contacts ***** 
    7: Get Contact
    8: Add Contact
    9: Remove Contact
